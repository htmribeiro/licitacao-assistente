# Requisitos do Projeto

## Funcionalidades Principais
1. Upload de arquivos de editais de licitação.
2. Interpretação automática utilizando NLP.
3. Geração de resumos dos editais.
4. Interface amigável para visualização.

## Tecnologias Sugeridas
- **Backend**: FastAPI, SpaCy
- **Frontend**: React.js
- **Modelos de NLP**: OpenAI GPT, Hugging Face
